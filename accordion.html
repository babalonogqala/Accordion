JavaScript is the language of interactivity on the web. It lets you respond to user actions, manipulate the DOM, fetch data from APIs, run animations, validate forms, and build full-scale applications — all directly in the browser without a page reload.

The Document Object Model (DOM) is a tree-like representation of your HTML that the browser builds when a page loads. JavaScript can read and change any node in that tree — adding elements, updating text, changing styles — which is how dynamic UIs are built without full page refreshes.

var is function-scoped and hoisted, which can cause surprising bugs. let is block-scoped and reassignable — the modern default for variables that change. const is also block-scoped but cannot be reassigned after declaration, making it ideal for values that should remain fixed.

Event listeners let you attach a callback function to any DOM element that fires when a specific event occurs — clicks, key presses, mouse movements, form submissions, and more. You attach them with element.addEventListener('event', callback) and remove them with removeEventListener when they're no longer needed.

The == operator checks for equality after performing type coercion, so 0 == false evaluates to true. The === operator is strict — it checks both value and type with no coercion, so 0 === false is false. Always prefer === to avoid subtle comparison bugs in your code.
